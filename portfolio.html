Reading is Fun!

Technologies Used: JQuery, Bootstrap , Google Books API, Giphy API, Full Calendar Library Plugin , Moment.js, Google Firebase

This was the first group project I completed during a JavaScript bootcamp. Our group created an app that allows teachers to create a reading list for students. The app has two main pages: the teacher portal and the student portal. Teachers can search for books, add to a reading list, and use a calendar to assign books. Students can request books and track their progress with stickers.
